  • jest 测试
  • typedoc 生成文档
  • 支持 Tree-shaking

Tree-shaking 说明

package.json 中加入 module sideEffects ，在生产环境构建时将启用 Tree-shaking 特性，减少包体积。

  "module": "es/index.js", // webpack resolve.mainfields 主入口默认为 ['browser', 'module', 'main']
  "sideEffects": false, // 标识副作用的文件，让 webpack 不要去除
