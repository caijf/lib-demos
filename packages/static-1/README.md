# 静态资源打包

gulp 处理 html、js、css、image 等静态资源打包。

适用于一些静态页面管理，如协议页面。

## 打包

```bash
npm run build
```

## 注意事项

- 文件路径和名称不变
- `html` 中使用的资源请通过相对路径
- 支持 `es6` 转 `es5`，但是因为没有处理模块合并，所以不要有模块相互依赖。建议只使用 es6 新增的语法
- 不支持 ts，因为文件路径和名称都不变，主要处理压缩
- 如果要更新文件，最好是新增不同版本的文件，避免覆盖和缓存问题。如 `jquery-3.6.5.js` `zepto-1.js` `header-merchant-destroy@200x200.jpg`
